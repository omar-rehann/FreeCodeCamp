Introduction

JavaScript is a programming language used for web development.

It can run in the browser or on the server.

console.log("Hello World");

It is versatile and widely used.

  • Client-side scripting
  • Server-side scripting
Variables

Variables are containers for storing data values.

We can declare variables using var, let, or const.

let name = "Omar";

Use const for values that will not change.

  • var (old way)
  • let (block scoped)
  • const (constant)
Functions

Functions are blocks of code designed to perform a particular task.

They are executed when called (invoked).

function greet() { console.log("Hello"); }

Functions can return values.

  • Function Declaration
  • Function Expression
Loops

Loops execute a block of code a number of times.

JavaScript supports for, while, and do...while loops.

for(let i=0; i<5; i++){ console.log(i); }

Loops can be nested.

  • for loop
  • while loop
  • do...while loop
Objects

Objects are collections of key-value pairs.

They are used to store multiple values in a single variable.

let person = {name: "Omar", age: 25};

Objects can contain methods.

  • Properties
  • Methods
